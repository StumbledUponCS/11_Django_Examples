VS Code

Getting Started

    open the Command Palette :


  1. By Pressing Shift, Command and P


  2. (Shift + Cmnd + P)


    Search & Start Terminal: Create New Integrated Terminal :


  1. The terminal panel will open at the bottom








Creating A New Virtual Environment

Creating A New Virtual Environment

    In the terminal run the commands :


  1. python3 -m venv env


  2. source env/bin/activate


    A new virtual environment called env is now created for the workspace :


  1. To see this, go to the explorer panel and expand the newly created folder under the workspace


  2. This is picture in this image








Selecting A Virtual Environment

Selecting A Virtual Environment

    Open the Command Palette again :


  1. search and select Python: Select Interpreter


  2. Now you will see the newly created virtual environment


    1. (something similar to Python 3.7.4 64-bit (‘env’, venv))

    2. Select the virtual environment as your interpreter

  3. Virtual environment is ready








Integrated Terminal

    Integrated Terminal :


  1. Starting a project, running the server & all other commands can be done using the virtual environment’s Integrated Terminal


    Once You're done:


  1. Close the web server :


    1. To close the web server go to the terminal and press CONTROL-C

Creating A Launch Profile

    A Launch Profile :


  1. With A Launch Profile :


    1. You don't need to type a command in the terminal every time you start the python django web server

    2. part of the the debugger settings in VS Code







Creating A Launch Profile

    A Launch Profile :


  1. Go to VS Code’s debug view by pressing the debug icon at the left activity panel


    1. Press the settings icon (gear icon) with a warning dot at the top, next to the No Configuration







Creating A Launch Profile

    Pressing the settings icon :


  1. a prompt box opens at the top with the list of debugger options


  2. From the list select Django: Launch and debug a Django web application








Creating A Launch Profile

    launch.json :


  1. A new file called launch.json will be created under the .vscode folder and opens in the editor window


    1. This file will have the debug configurations

  2. Make sure the content of the file is similar to the image


  3. Here the values in the program and the args section performs similar to the terminal command python3 manage.py runserver –noreload


  4. Save the file








Creating A Launch Profile

    Python: Django debug configuration :


  1. check the debug configuration drop down list at the top left


    1. Python: Django will be selected by default

    2. If not choose it manually







Creating A Launch Profile

    The launch profile setting is now complete :


  1. You can now start the web server and the debugger just by pressing the green arrow found in between DEBUG and Python: Django


  2. Other ways to start the web server in debug mode :


    1. by pressing F5 key

    2. by going to the menu >> Debug >> Start Debugging







Creating A Launch Profile

    The web server is now started in debug mode :


  1. You can identify the changes happened in the Visual Studio Code editor


    1. A debugging toolbar appears at the top

    2. The color of the status bar changes to orange







Creating A Launch Profile

    To open the web app in the browser :


  1. either command+click the url http://127.0.0.1:8000/


  2. copy and paste it to the browser




  3. NOTE: If you want to start the web server without debug mode, then press the Control + F5 keys





  4. Once done, don’t forget to close the web server by pressing CONTROL-C
