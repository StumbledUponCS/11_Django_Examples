Adding Static Files To Django

    Adding Images :


  1. Open the urls.py folder under the web app folder


  2. Add the statement to staticfiles_urlpatterns if it is not already available

    1. from django.contrib.staticfiles.urls import staticfiles_urlpatterns


  3. Add this line to the end of the same urls.py file if it is not already available

    1. urlpatterns += staticfiles_urlpatterns()



  4. Here is the sample urls.py in the web app. In this, my web app is named as dj_app









Adding Images :

    Adding Images :


  1. Create a folder for images

    1. Here, I have named the folder as media
      under the web app’s static folder : static/dj_app



  2. Add the image to this new folder static/dj_app/media.







Adding Images :

    To add the image file to a HTML template :


  1. Open a template

  2. Add the the HTML img tag along with {% load static %} &
    {% static “image file url” %} placeholders, like this :

    1. {% load static %} img src=""

    2. {% static "dj_app/media/house.jpg" %}" alt="home" />
