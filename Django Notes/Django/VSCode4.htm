Creating And Using A Base Template :

    Multiple Pages :


  1. In most of the web applications there will be multiple pages

  2. Some of the content in all the pages will be common for all the pages

  3. For example the navigation menu will be common to all the pages

  4. So, it is always a good practice to have these type of common content in a single file

    1. It is also easy to maintain







Creating And Using A Base Template :

    Multiple Pages :


  1. In Django web app, the base template file will have the common content pertinent to several templates and views

  2. Let us see how and where to crate a base template and refer it to other templates or views
